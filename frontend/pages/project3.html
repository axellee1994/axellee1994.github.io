FT_TRANSCENDENCE

Overview

42 Transcendence is a full-stack web application featuring a multiplayer Pong game, user authentication, real-time chat, and social features.

Technologies Used

Backend:
Fastify TypeScript Node.js SQLite
Frontend:
TypeScript Webpack Tailwind CSS Babylon.js
DevOps:
Docker

Features

  • Real-time Multiplayer Pong: Challenge friends or other users to a classic game of Pong.
  • User Authentication: Secure user registration and login using JWT and Google OAuth2.
  • Social Features: Add friends, view user profiles, and see their match history.
  • Real-time Chat: Communicate with other players in real-time.
  • Tournament Mode: Compete in tournaments to become the ultimate Pong champion.

What I learnt

  • Frontend Architectures: Learning how to build the frontend with MCV structure to be connected to the backend.
  • Stress Testing: Different types of testing to find bugs and resolve it during UAT, UAP and SLA.
  • Developed user management and authentication system: Helped to implement JWT tokens for authentication in order to retrieve data for social features and detailed user profiles with performance tacking.
  • Building interactice dashboards: Used data visualization (charts and graphs) to provide clear, actionable insights into the statistics.
  • Implementation of AI-drive opponent: How to replicate human behavior by simulating keyboard inputs, implementing decision-making logic and strategies.
