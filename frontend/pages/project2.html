Computer Vision Model

Overview

An end-to-end multi-class image classifier using TensorFlow 2.0 and TensorFlow Hub

Goal

  • Problem Definion: Identifying the breed of a dog given an image of a dog.
  • Data: Dataset came Kaggle's dog breed identification competition
  • Evaluation: A file with prediction probabilitirs for each dog breed of each test image
  • Features: Dealing with images(unstructured data), 120 breed of dogs and 10,000+ images per set

Technologies Used

Language:
Python
ML & Data:
NumPy Pandas scikit-learn TensorFlow 2.0
Modeling:
MobileNet_V2_130_224
Viz & EDA:
seaborn matplotlib
Tools:
Google Colab SciPy Docker GitHub

Result Summary

  • Best Model: Transfer learning with MobileNetV2 (TensorFlow Hub)
  • Test Accuracy: Achieved up to ~65% on a subset (1000 images)
  • Recall Score: Model learns directly from image pixels
  • Primary Signal: Predictions with probability columns for each breed

What I learnt

  • Exploratory Data Analysis: Cleaning model by replacing null values or dropping duplicates before using crosstabs and heatmaps
  • Stratified Data Splitting: How to determine the split percentages and to use train_test_split
  • Comparison among models: Learning how to tune with different models before doing a final tune with GridSearchCV
  • Pipeline: Implementing a pipeline to repeat scaling and preprocessing
  • Evaluation beyong accuracy: Cross validation with multiple scorers, ROC curve, ROC-AUC and Confusion Matrix
  • Feature Importance: Using .feature_importances_ to see which features affects the tuning most.
  • Model Export: Importing joblib and export a tuned model for future use
